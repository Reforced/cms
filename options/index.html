Chillout Music Script - Options

Autovote

Automatically updubs current song.

Auto Clear Chat

Automatically clears the chat every 30 minutes.

Work Mode

Disables the video, options and the chat.

Chat Mode

Disables the video, options and makes chat bigger.

Hide Deleted Messages

Hides messages that have been deleted.

Community Theme

Enables the room's custom theme set in the description.

Join Message

Displays who joined the room in chat.

Leave Message

Displays who leaves the room in chat.

Grab Message

Displays who grabs the song in chat.

Up Dub Message

Displays who updubs the song in chat.

Custom Background

Allows you to set a custom background with an image link.

Custom Css

Allows you to set a custom theme (css) with a css file link.
